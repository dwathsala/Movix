# 🎬 Movie App (React Native + Expo)

A modern Movie App built using **React Native (Expo)** that allows users to search movies, view details, and explore trending content. This app integrates with external APIs and uses Appwrite for backend services.

---

## 🚀 Features

- 🔍 Search movies in real-time
- 🎥 View detailed movie information
- 📊 Track trending searches
- ⭐ Display ratings, votes, and runtime
- 🔥 Trending movies section
- ✔️ Save movies
- ⚡ Smooth and responsive UI

---

## 🛠️ Tech Stack

- **Frontend:** React Native (Expo)
- **Navigation:** Expo Router
- **Backend:** Appwrite
- **API:** TMDB (The Movie Database)
- **State/Data Handling:** Custom Hooks (`useFetch`)
- **Styling:** Tailwind CSS (NativeWind)

---

## 🔗API Used
🎬 The Movie Database (TMDB) - https://www.themoviedb.org/
